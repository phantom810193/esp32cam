快速驗收步驟

  1. 啟動後端伺服器，確認 http://127.0.0.1:8000 可連線。
  2. 分別執行：curl -fsS "http://127.0.0.1:8000/ad/ME0001/compose" | jq . 等指令。
  3. 確認 /srv/esp32-ads/out 內產生對應 JPG。

若要重新產生素材，可在 API 加上 ?force=1 或使用 CLI： python -m backend.tools.generate_ad --member ME0001 --force
